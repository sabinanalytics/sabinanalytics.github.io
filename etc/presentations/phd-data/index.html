The ideal of automation

Tell the computer what to do

Makefile = a list of instructions for the computer


output: required files
	commands to generate the output using the required files

Type make at the terminal to generate all output files.


Make keeps track of file timestamps;
no unnecessary file creation!

For example...


Table_1.doc: run_model.do
	@echo "Creating table..."
	stata-se -b do run_model.do

Or...


Output/figure_1.pdf: R/figures.R R/load_data.R Output/all_variations.csv
	@echo "Creating figures..."
	cd R; Rscript figures.R

Mix and Match!

You can use any program you want.

If you can type a command in the terminal, Make can do it.
